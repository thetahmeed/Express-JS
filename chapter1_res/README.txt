Trying express JS another time. Lets start

1. install npm by "npm init -y"
2. install express by "npm i express"
3. create a js file name "app.js"
4. modify the "package.json" file. Change main "index.js" to "app.js", add a start script "node app.js"
5. follow "app.js" for rest

Optional: "npm i -D nodemon" and change the ""start": "nodemon app.js" and it will run the app automatically
